Important HTML tags

This page is designed as quick reference for some common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document tags

Tag Name Purpose
DOCTYPE Document type declaration (just, don't worry about it)
html html opens and closes the entire html document
head Document header Header container that contains metadata about the website
body Body data for website Contains all visual data for the website
title Document title Title of the webpage which is provided to the browser tab
meta Meta A void element that tells the webpage which type of character set to use

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table date cell
header header make a page header
div division make a division in HTML
span span a generic container for phrasing content(?)
ol ordered list creates lists that are ordered with numbers
ul unordered list creates lists that are bulleted

Inline Elements

Tag Name Purpose Example Result
em empasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>Matt is boss</strong> Matt is boss
a anchor make hyperlink <a href="https://mgregg2.github.io/sample_website/">Matt's awesome site</a> Matt's awesome site
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code text to indicate code <code>strong</code> stong
li list element create a new list element for either ordered or unordered lists <li>list item</li>
